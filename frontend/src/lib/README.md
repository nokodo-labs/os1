# Liquid UI Components

Beautiful, modern UI components with a "liquid glass" aesthetic for nokodo AI.

## Philosophy

The Liquid UI design system combines:

- **Liquid Glass**: Transparent elements with light refraction effects at edges
- **Liquid Metal**: Mercury-like metallic effects with shimmer animations
- **Physics-based Animations**: Smooth, realistic interactions using cubic-bezier easing
- **Premium Feel**: Apple-inspired, fluid, reactive experiences
